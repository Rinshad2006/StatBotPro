 📊 StatBotPro – AI CSV Data Assistant

StatBotPro is an AI-powered data analysis assistant built with Python.
It allows users to analyze CSV datasets using natural language questions.
The system uses LangChain agents, Pandas, and visualization tools to automatically process and analyze data.

⸻

🚀 Features
	•	Ask natural language questions about CSV datasets
	•	Automatic data analysis using Python and Pandas
	•	Chart generation from dataset queries
	•	Secure execution of AI-generated code
	•	Interactive web interface using Streamlit
